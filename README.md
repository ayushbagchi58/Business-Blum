# Business Blum

A modern business funding platform built with Next.js, TypeScript, Tailwind CSS, and Framer Motion created by Ayush Bagchi.

## GitHub Profile: https://github.com/ayushbagchi58

## Features

- Responsive modern UI
- Animated sections with Framer Motion
- Mobile-friendly navigation
- Loan application pages
- CTA sections
- Timeline animations
- Optimized Next.js app structure

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Infinanze-Technologies/businessblum.com.git>
```

Go to the project folder:

```bash
cd your-project-name
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

## Build for Production

```bash
npm run build
npm start
```
